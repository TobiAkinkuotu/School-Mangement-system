# EduStream: Lightweight School Management System

A modern, high-performance school management system built with **FastAPI**, **SQLAlchemy**, and a premium **Glassmorphism UI**.

## ✨ Features
- **Modern Tech Stack**: Python 3.14 compatible, FastAPI, SQLAlchemy 2.0.
- **Premium Design**: Sleek, responsive, and animated user interface.
- **Core Modules**: Dashboard, Student enrollment, Teacher management, Attendance tracking, and Academic grading.
- **Role-Based Views**: Tailored experiences for Admins and Students.

## 🚀 Quick Start
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python app.py
   ```
4. Access at [http://127.0.0.1:8000](http://127.0.0.1:8000) (Admin: `admin` / `admin123`).

## 🛠️ Tech Stack
- **Backend**: FastAPI
- **Database**: SQLite (SQLAlchemy 2.0)
- **Frontend**: Jinja2 + Vanilla CSS (Premium Glassmorphism)
- **Auth**: Bcrypt
